## Core Responsibilities by Role

### **Member 1: Computational Biology Lead**
- **Primary Phases:** Primary Analysis (QC/preprocessing), Secondary Analysis (alignment)
- **Core Skills:** Pipeline design, bioinformatics tools (FastQC, Bowtie2, TopHat), version control, scripting
- **Deliverables:** Clean preprocessing code, alignment results, reproducible GitHub repo, Methods (computational section)
- **Weekly Reports Focus:** Infrastructure decisions, code quality, pipeline blockers, data handoff documentation

### **Member 2: Statistical Analysis & Fairness Audit Lead**
- **Primary Phases:** Tertiary Analysis (DE analysis), Fairness auditing (Weeks 11-15)
- **Core Skills:** Statistical methodology, fairness metrics (disparate impact, equalized odds), visualization, multiple testing correction
- **Deliverables:** DE results with statistics, fairness audit reports, publication-quality visualizations, Methods (statistical section)
- **Weekly Reports Focus:** Statistical decisions, fairness findings, validation approaches, visualization development

### **Member 3: Biological Integration & Literature Lead**
- **Involved in:** All phases (biological lens throughout)
- **Core Skills:** Literature synthesis, biological interpretation, pathway analysis, functional validation
- **Deliverables:** Literature synthesis (15+ sources), biological interpretation, Introduction/Discussion sections, paper coherence
- **Weekly Reports Focus:** Literature progress, biological interpretation, writing status, data interpretation support
